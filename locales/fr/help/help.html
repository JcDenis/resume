Menu simple

Le plugin Simple Menu est utilisé par le thème pour afficher le menu latéral.

Astuce : pour ajouter un lien vers le bloc des widgets, vous pouvez créer un élément de menu libre avec un lien vers #menubar

Image de profil

Vous pouvez remplacer l'image de la barre latérale par une image de votre médiathèque. Cliquez sur le bouton Changer, choisissez l'image, puis enregistrez. Cliquez sur le bouton Réinitialiser pour utiliser l'image par défaut.

Couleur principale

Choissez votre couleur principale et enregistrez. Cette couleur est utilisée pour la barre latérale, les liens et les boutons. La valeur par défaut est #bd5d38

Liens vers les médias sociaux

Les étiquettes s'afficheront en haut de l'écran. Vous devez indiquer pour chacune un libellé et l'URL correspondante. (Laissez l'un de ces deux champs vide pour ne pas afficher l'étiquette).

Vous pouvez également modifier l'ordre d'affichage des étiquettes par tiré-lâché si javascript est actif et que vous ne l'avez pas interdit dans vos préférences. Si javascript est désactivé, un champ supplémentaire permet d'indiquer l'ordre d'affichage (de 1 à n).
