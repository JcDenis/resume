Simple Menu

Simple Menu plugin is used by the theme to manage the sidebar menu.

Tip : to add a link to the widgets block, create a free menu item with a link to #sidebar

Profile image

You can replace the left sidebar image by your own from your media manager. Click on Change button to choose the image and save your changes. If you wish to use default profile image, click on Reset button.

Main color

Choose a color for sidebar, links and buttons, and save. Default color is #BD5D38

Social links

Stickers will be displayed at the top of the screen. For each, you must enter a label and the corresponding URL. (Leave one of these fields empty to avoid the sticker display).

You may change the stickers display order by grabbing and moving their lines if javascript is enabled and allowed in your preferences. If javascript is disabled, a field allows to indicate the stickers' display order (from 1 to n).
